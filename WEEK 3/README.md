# 守护进程
## 功能
检查服务程序是否死机或挂起，如果是，则杀死服务程序
服务程序被杀死后，调度程序（procctl）会重新启动服务程序
## 判断方式
心跳机制：
1. 服务程序在共享内存中创建心跳结构体并更新
2. 守护程序每隔一段时间对共享内存进行遍历，杀死超时程序
## exit函数
exit函数不会调用局部对象构造函数
exit函数会调用全局对象析构函数
return会调用局部和全局对象析构函数
